# Trance Releases Weekly

A weekly updated repository tracking new trance music releases. Every Monday, a fresh list of tracks, albums, EPs, and remixes is posted to help you stay up to date with the latest sounds from across the trance scene.

## How It Works

* **Update Schedule:** New lists are published at the beginning of every week (Monday).
* **Scope:** Covers various subgenres, including Uplifting, Progressive, Psytrance, Tech Trance, and Vocal Trance.
* **Format:** Each weekly list is organized in markdown, making it easy to browse, search, and link directly to streaming or purchase platforms (e.g., Beatport, Spotify, Bandcamp).

## Repository Structure

Releases are organized by year and week for easy navigation.
